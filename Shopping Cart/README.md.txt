I have made a Shopping Website with Jvascript.

Functionalities:
 
User can search product by productname, colour or productlook. 
User can see products within range  by giving minimum and maximum price.
User can add and remove products from cart and can see total products in cart.
Also a POP- UP page is made if products are not available.
